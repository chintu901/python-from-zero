## Numaric Data Type

Numeric data types in Python are used to **store numbers and perform mathematical operations**. Python mainly provides **three numeric data types**.

### 1. Integer (`int`):

Integers are whole numbers with out any decimal point. They can be **positive, negative, or zero**.

Example:

``` python
a = 5
b = 8

print(a + b) # Addition 
print(a - b) # Subtraction
print(a * b) # Multiply
print(a // b) # Integer Division
```

### 2. Floating Point Number (`float`):

Float data types are used to store **decimal numbers**. They can be positive or negative.

``` python
a = 10.0
b = 2.5

print(a / b) # Division
```

> [!WARNING]
> Floating-point numbers may somtime have **precision issue**

### 3. Complex Numbers (`complex`):

Complex numbers consist of a **real par**t and an **imaginary part** and are written using `j` for the imaginary unit.

Example:

``` python
x = 2 + 3j

# Accessing parts
print(x.real) # Output: 2.0 
print(x.imag) # Output: 3.0
```

**Key Points:**

- `int` → whole numbers

- `float` → decimal numbers

- `complex` → real + imaginary numbers

Numeric data types are essential for **calculations**, **data processing**, and **logic building** in Python.
