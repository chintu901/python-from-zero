# Mapping Data Type

Mapping data types in Python are used to **store data in key–value pairs**.
Instead of accessing elements using an **index** (like lists or tuples), mapping data types allow you to access values using **keys**.

The main mapping data type in Python is the **dictionary** (`dict`).

**Key Features of Mapping Data Types:**

- Data is stored as key : value pairs
- Keys are used to access values
- Keys must be unique and immutable
- Values can be of any data type

> [!NOTE]
> Mapping data types are best when you need fast access to data using meaningful names.

## Dictionary (`dict`) in Python

A dictionary is a built-in Python data type used to **store and manage data in key–value pairs**.

**Dictionary Syntax:**

``` bash
dictionary_name = {key1 : value1, key2 : value2}
```

**Example:**

``` python
person = {
    "name" : "Mohit",
    "city" : "Delhi",
    "age" : 20
}

print(person)   # {'name': 'Mohit', 'city': 'Delhi', 'age': 20}
```

![Dictionary diagram](https://pythontic.com/DictionaryViews_Python.png)

**Characteristics of Dictionaries:***
- Unordered in older Python versions (ordered from Python 3.7+).
- Mutable (values can be changed).
- Keys must be unique.
- Keys can be of types like `int`, `str`, or `tuple`.

**Accessing Dictionary Value:**

``` python 
print(person["name"])   # Mohit
```

> [!NOTE]
> Dictionaries allow you to store **structured**, **real-world data** efficiently using key–value pairs.

## Creating Dictionaries

In Python, dictionaries can be created in **multiple ways**, depending on your needs.

**Using Curly Braces `{}` (Most Common):**

``` python
student = {
    "name": "Alex",
    "age": 20,
    "course": "Python"
}
```

**Using the `dict()` Constructor**

``` python
student = dict(name = "Alex", age = 20, course = "python")

print(student)  # {'name': 'Alex', 'age': 20, 'course': 'python'}
```

> Note: Keys must be **unique and immutable**, while values can be of any data type.

## Accessing Dictionary Elements

Dictionary elements are accessed using keys, not indexes.

**Accessing Valuse Using keys**

``` python
student = {
    "name": "Alex",
    "age": 20
}

print(student["name"])  # Alex
```

**Using the `get()` Method**

``` python
print(student.get("age"))       # 20
print(student.get("grade"))     # None
```
- `get()` does not raise error if the key is missing
- you can provide a default value

``` python
print(student.get("grade", "Not available"))    # Not available
```

> [!NOTE]
> Use `get()` when you are not sure whether a key exists.

## Adding and Updating Dictionary Elements

Dictionaries are mutable, so you can add and update elements after creation.

**Adding a New Key: Value Pair:**

``` python
person = {"name": "Jon"}

person["age"] = 20

print(person)   # {'name': 'Jon', 'age': 20}
```

**Updating an Existing Value:**

``` python
person["age"] = 21  # Change the value from 20 to 21 
```

**Using `update()`:**

``` python
student.update({"course": "python", "year": 2026})
```

> Note: Dictionaries make it easy to add, modify, and manage structured data efficiently.

## Removing Elements from a Dictionary

Python provides multiple ways to **remove key–value pairs** from a dictionary.

### Using `del` Keyword

Removing a specific key - value pairs.

``` python
student = {
    "name": "Alex",
    "age": 20
}
del student["age"]
print(student)  # {'name': 'Alex'}
```

> [!WARNING]
> If the key does not exist, it raises a KeyError.

### Using `pop` Method

Removes a key and returns its value.

``` python
student = {
    "name": "Alex",
    "age": 20
}

age = student.pop("age")
print(age)      # 20
print(student)  # {'name': 'Alex'}
```

You can also provide a default value to avoid errors.

``` python
student.pop("age", "Key not found")
```

### Using `popitem()` Method

Removes and returns the **last inserted key–value pair** (Python 3.7+).

``` python
student = {"name": "Alex", "age": 20}
student.popitem()
```

### Using `clear()` Method

Remove all elements from the `dict`.

``` python
student.clear()
```

> [!NOTE]
> Use `pop()` when you need the removed value, and `del` when you just want to delete.

## Dictionary Methods

Python dictionaries come with useful built-in methods to **work with keys and values**.

| Method | Description | Example |
|-------|-----|----------|
| `copy()` | Returns a shallow copy of the dictionary | `	d.clear() ` |
| `fromkeys(seq, value)` | Creates a new dictionary from a sequence of keys with the same value | `dict.fromkeys(['a','b'], 0) ` |
| `items()` | Returns a view object of (key, value) pairs | `d.items() ` |
| `keys()` | Returns a view object of dictionary keys | `d.keys()` |
| `setdefault(key, default)` | Returns value if key exists; otherwise inserts key with default value | `d.setdefault('x', 0)` |
| `values()` | Returns a view object of dictionary values | `dict,values()` |

**Example Program:**

``` python
# Demonstrating Python dictionary methods

# Initial dictionary
person = {"name": "Alice", "age": 25, "city": "New York"}

# copy()
person_copy = person.copy()

# setdefault()
person.setdefault("email", "alice@example.com")

# keys(), values(), items()
keys_list = list(person.keys())
values_list = list(person.values())
items_list = list(person.items())

# Output results
print("Updated person:", person)
print("Keys:", keys_list)
print("Values:", values_list)
print("Items:", items_list)
print("Cleared copy:", person_copy)
```
**Output:**

``` bash
Updated person: {'name': 'Alice', 'age': 25, 'city': 'New York', 'email': 'alice@example.com'}
Keys: ['name', 'age', 'city', 'email']
Values: ['Alice', 25, 'New York', 'alice@example.com']
Items: [('name', 'Alice'), ('age', 25), ('city', 'New York'), ('email', 'alice@example.com')]
Cleared copy: {'name': 'Alice', 'age': 25, 'city': 'New York'}
```

> [!NOTE]
> Dictionary methods make data **access**, **updates**, and **iteration** clean and efficient.
