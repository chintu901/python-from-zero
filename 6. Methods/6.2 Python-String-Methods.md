# Python String Methods

In Python, a string is a data type used to store **textual data**, such as words, sentences, numbers written as text, or symbols. Strings are written inside quotes—either single (`' '`) or double (`" "`).

``` python
text = "Python"
```

Because text is such a common form of data, Python provides **special methods** to work with strings easily and safely.

## Common transformation methods:

| Methods | Description | Example |
|-------|-----|----------|
| `capitalize()` |  First character uppercase, rest lowercase. | `text.capitalize()` |
| `upper()` | Converts text to uppercase. | `text.upper()` |
| `lower()` | Converts text to lowercase. | `text.lower()` |
| `casefold()` | Converts text to lowercas and can handle more characters and is more effective in case-insensitive comparisons. | `text.casefold()` |
| `title()` | Capitalizes first letter of each word. | `text.title()` |
| `swapcase()` | Swaps uppercase to lowercase and vice versa. | `text.swapcase()` |
| `strip()` | Removes leading and trailing characters from a string. By default removes "whitespaces" but also can remove string of characters | `text.strip('#')` |
| `lstrip()` | Removes the leading characters fro the string. By default removes "whitespaces". | `text.lstrip()` |
| `rstrip()` | Remove trailing characters (characters at the end of a string). By default removes "whitespaces". | `text.rstrip()` |
| `replace(old, new)` | Replace all occurrences of a substring. | `text.replace('c','a')` |
| `zfill(width)` | Pad with zeros on the left. | `text.zfill(5)` |

**Example:**

``` python
# String Transformation Methods Demo

text = "  hello PyTHon world  "

# capitalize()
print("capitalize():", text.capitalize())

# casefold(), lower(), upper()
print("casefold():", text.casefold())
print("lower():", text.lower())
print("upper():", text.upper())

# title()
print("title():", text.title())

# swapcase()
print("swapcase():", text.swapcase())

# strip(), lstrip(), rstrip()
print("strip():", text.strip())
print("lstrip():", text.lstrip())
print("rstrip():", text.rstrip())

# replace(old, new)
print("replace():", text.replace("PyTHon", "Python"))

# zfill(width)
number = "123"
print("zfill():", number.zfill(6))
```

> [!NOTE]
> Common transformation methods in Python are used to change the appearance or structure of strings without modifying the original data.

## Search and check methods:

| Methods | Description | Example |
|-------|-----|----------|
| `find` | Find first occurrence of a specified substring within a string and return its index. | `text.find('car')` |
| `rfind()` | Find the last (rightmost) occurrence of a specified substring within a string and return its index.  | `text.rfind('truck')` |
| `index()` | Find the first occurrence of a specified element in a list or string. If the element is not found, it raises a `ValueError`. | `text.index('banana')` |
| `rindex()` | Find the last (rightmost) occurrence of a specified in a list or string. If the element is not found, it raises a `ValueError`. | `text.rindex('banana')` |
| `startswith()` | Checks if a string begins with a specified prefix and returns a boolean result. optional start and end parameters. | `text.startswith('hello', 2, 20)` |
| `endswith()` | Checks if a string ends with a specified suffix and return a boolean result. | `endswith('world', 5, 9)` |
| `count()` | Count occurrences of a substring. | `text.count('c')` |

**Example:**

``` python
# Search and Check String Methods Demo

text = "python programming is fun with python"

# find() and rfind()
print("find('python'):", text.find("python"))     # first occurrence
print("rfind('python'):", text.rfind("python"))   # last occurrence

# index() and rindex()
print("index('programming'):", text.index("programming"))
print("rindex('python'):", text.rindex("python"))

# startswith() and endswith()
print("startswith('python'):", text.startswith("python"))
print("endswith('python'):", text.endswith("python"))

# count()
print("count('python'):", text.count("python"))
```

> [!NOTE]
> Search and check methods in Python are used to locate text, verify patterns, and analyze string content.

## Validation and Checking Methods:

| Methods | Description | Example |
|-------|-----|----------|
| `isalnum()` | Check whether all characters in a string are alphanumeric. | `text.isalnum()` |
| `isalpha()` | Checks if all characters in a string are alphabetic (a-z, A-Z). | `text.isalpha()` |
| `isdigit()` | Checks whether all characters in a string are digits (0–9). | `text.isdigit()` |
| `isnumeric()` | Checks whether all characters in a string are numeric characters. | `text.isnumeric()` |
| `isspace()` | Checks if all characters in a string are whitespace characters. | `text.isspace()` |
| `isidentifier()` | checks whether a string is a valid identifier. An identifier is a name given to entities like variables, functions, classes, etc. | `text.isidentifier()` |
| `isprintable()` | Checks whether all characters in a string are printable (including spaces). | `text.isprintable()` |
| `isascii()` | Checks whether all characters in a string are ASCII characters (code points 0–127). | `text.isascii()` |
| `islower())` | Checks whether all characters in a string are lowercase. | `text.islower()` |
| `isupper()` | Checks whether all characters in a string are uppercase. | `text.isupper()` |
| `istitle()` | Checks if a string is in title case, meaning each word starts with an uppercase letter followed by lowercase letters.  | `text.istitle()` |

**Example:**

``` python
# Validation and Checking String Methods Demo

# Sample strings
text_alpha = "Python"
text_alnum = "Python123"
text_digit = "12345"
text_space = "   "
text_identifier = "variable_name"
text_printable = "Hello!\n"
text_ascii = "Hello123"
text_title = "Python Programming"
text_lower = "python"
text_upper = "PYTHON"

# Character property checks
print("isalnum():", text_alnum.isalnum())
print("isalpha():", text_alpha.isalpha())
print("isdigit():", text_digit.isdigit())
print("isnumeric():", text_digit.isnumeric())
print("isspace():", text_space.isspace())
print("isidentifier():", text_identifier.isidentifier())
print("isprintable():", text_printable.isprintable())
print("isascii():", text_ascii.isascii())

# Case checking methods
print("islower():", text_lower.islower())
print("isupper():", text_upper.isupper())
print("istitle():", text_title.istitle())
```

> [!NOTE]
> Validation and checking string methods are used to verify the nature and correctness of text data before processing it further. These methods return Boolean values (True or False), making them especially useful in conditions and input validation.

## Splitting and joining:

| Methods | Description | Example |
|-------|-----|----------|
| `split()` | Used to break a string into a list of substrings based on a specified separator and you can aslo control max split. | `text.split(" ", 2)` |
| `rsplit()` | Split from the end, especially useful when only the last few splits matter | `text.rsplit(" ", 2)` |
| `rsplit()` | Split from the end, especially useful when only the last few splits matter | `text.rsplit(" ", 2)` |
| `splitlines()` |  | `text.` |
| `splitlines()` | Used to split a string into a list of lines based on line breaks. | `text.splitlines()` |
| `partition()` | Split a string into exactly three parts based on the first occurrence of a specified separator. | `text.partition()` |
| `rpartition()` | Split a string into three parts based on the last occurrence of a specified separator. | `text.rpartition("-")` |
| `join()` | Combine elements of an iterable (like a list, tuple, or set) into a single string, with a specified separator between each element. | `sentence = " ".join(words)` |

**Example:**

``` python
# Splitting and Joining String Methods Demo

text = "python,java,c++,javascript"
multiline_text = "Python\nJava\nC++\nJavaScript"

# split() and rsplit()
print("split():", text.split(","))
print("rsplit():", text.rsplit(",", 2))

# splitlines()
print("splitlines():", multiline_text.splitlines())

# partition() and rpartition()
sentence = "Learn Python Programming"
print("partition():", sentence.partition("Python"))
print("rpartition():", sentence.rpartition("Python"))

# join()
languages = ["Python", "Java", "C++", "JavaScript"]
joined_text = " | ".join(languages)
print("join():", joined_text)
```

> [!NOTE]
> Splitting and joining string methods are used to break text into smaller parts or combine multiple elements into a single string. These operations are essential when working with structured text such as CSV data, user input, or multiline content.

## Formatting and translation:

| Methods | Description | Example |
|-------|-----|----------|
| `format()` | create formatted strings by inserting values into placeholders {} inside a string. | `string.format(value1, value2, ...)` |
| `format_map())` | Format strings by replacing placeholders with values from a dictionary. | `string.format_map(dic_name)` |
| `format()` | create a mapping table for character replacements. Often used with the translate() method  | `str.maketrans(x, y=None, z=None)` |
| `maketrans()` | Replace specific characters in a string based on a translation table.  | `str.maketrans(x, y=None, z=None)` |
| `translate()` | Replace specific characters in a string based on a translation table.  | `string.translate(table)` |
| `expandtabs()` | Replaces tab characters (\t) in a string with the appropriate number of whitespace characters, depending on the tab size specified.  | `text.expandtabs(4)` |

**Example:**

``` python
# Formatting and Translation String Methods Demo

# format() and format_map()
name = "Alex"
age = 22

formatted_text = "Name: {}, Age: {}".format(name, age)
print("format():", formatted_text)

data = {"lang": "Python", "level": "Beginner"}
formatted_map = "Language: {lang}, Level: {level}".format_map(data)
print("format_map():", formatted_map)

# maketrans() and translate()
text = "hello python"
translation_table = str.maketrans("hp", "HP")
translated_text = text.translate(translation_table)
print("translate():", translated_text)

# expandtabs()
tab_text = "Python\tProgramming\tLanguage"
print("expandtabs():", tab_text.expandtabs(4))
```

> [!NOTE]
> Formatting and translation methods are used to control how text appears and how characters are transformed in a string. Formatting methods help insert values into text in a structured and readable way, while translation methods allow bulk character replacement using predefined mapping rules.
