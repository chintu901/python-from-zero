# Built-in Functions for List

Built-in functions are **general Python functions** that work with lists to perform common operations like **counting elements**, **finding maximum** or **minimum values**, **sorting data**, and **calculating totals**. These functions are **not specific to lists only**, but they are commonly used with lists to analyze and process list data efficiently without writing extra code.

## Common Built-in Functions Used with Lists

Python provides several built-in functions that are very useful when working with lists. These functions help you analyze, manipulate, and process list data easily.

| Functions | Description | Syntax |
|-------|-----|----------|
| `len()` | Returns the number of elements in the list. | `print(len(list))` |
| `sum()` | Returns the sum of all numeric elements in the list. | `print(sum(list))` |
| `max()` | Returns the largest element in the list. | `print(max(list))` |
| `min()` | Returns the smallest element in the list. | `print(min(list))` |
| `sorted()` | Returns a new sorted list without changing the original list. | `print(sorted(list))` |
| `any()` | Returns `True` if at least one element is `True`. | `print(any(list))` |
| `all()` | Returns `True` if all elements are `True`. | `print(all(list))` |
| `enumerate()` | Returns index and value pairs, useful in loops. | `print(enumerate(list))` |

**Example:**

``` python
# List of numbers
numbers = [10, 20, 5, 0, 30]

# List of boolean values
bool_list = [True, False, True]

# 1. len() - Number of elements
print("Length of numbers list:", len(numbers))

# 2. sum() - Sum of all elements
print("Sum of numbers:", sum(numbers))

# 3. max() - Largest element
print("Maximum value:", max(numbers))

# 4. min() - Smallest element
print("Minimum value:", min(numbers))

# 5. sorted() - Returns a new sorted list
sorted_numbers = sorted(numbers)
print("Sorted list:", sorted_numbers)
print("Original list (unchanged):", numbers)

# 6. any() - True if at least one element is True
print("Any True in bool_list:", any(bool_list))

# 7. all() - True if all elements are True
print("All True in bool_list:", all(bool_list))

# 8. enumerate() - Index and value pairs
print("\nUsing enumerate():")
for index, value in enumerate(numbers):
    print(f"Index {index} -> Value {value}")
```
