# Python String Methods

In Python, a string is a data type used to store **textual data**, such as words, sentences, numbers written as text, or symbols. Strings are written inside quotesâ€”either single (`' '`) or double (`" "`).

``` python
text = "Python"
```

Because text is such a common form of data, Python provides **special methods** to work with strings easily and safely.

## Common transformation methods:

| Methods | Description | Example |
|-------|-----|----------|
| `capitalize()` |  First character uppercase, rest lowercase. | `text.capitalize()` |
| `upper()` | Converts text to uppercase. | `text.upper()` |
| `lower()` | Converts text to lowercase. | `text.lower()` |
| `casefold()` | Converts text to lowercas and can handle more characters and is more effective in case-insensitive comparisons. | `text.casefold()` |
| `title()` | Capitalizes first letter of each word. | `text.title()` |
| `swapcase()` | Swaps uppercase to lowercase and vice versa. | `text.swapcase()` |
| `strip()` | Removes leading and trailing characters from a string. By default removes "whitespaces" but also can remove string of characters | `text.strip('#')` |
| `lstrip()` | Removes the leading characters fro the string. By default removes "whitespaces". | `text.lstrip()` |
| `rstrip()` | Remove trailing characters (characters at the end of a string). By default removes "whitespaces". | `text.rstrip()` |
| `replace(old, new)` | Replace all occurrences of a substring. | `text.replace('c','a')` |
| `zfill(width)` | Pad with zeros on the left. | `text.zfill(5)` |

**Example:**

``` python
# String Transformation Methods Demo

text = "  hello PyTHon world  "

# capitalize()
print("capitalize():", text.capitalize())

# casefold(), lower(), upper()
print("casefold():", text.casefold())
print("lower():", text.lower())
print("upper():", text.upper())

# title()
print("title():", text.title())

# swapcase()
print("swapcase():", text.swapcase())

# strip(), lstrip(), rstrip()
print("strip():", text.strip())
print("lstrip():", text.lstrip())
print("rstrip():", text.rstrip())

# replace(old, new)
print("replace():", text.replace("PyTHon", "Python"))

# zfill(width)
number = "123"
print("zfill():", number.zfill(6))
```

> [!NOTE]
> Common transformation methods in Python are used to change the appearance or structure of strings without modifying the original data.
